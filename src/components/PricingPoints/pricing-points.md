# PricingPoints

## Descripción
Átomo para mostrar puntos a acumular o canjear junto al logo de Loyalty.

**Tipo:** Átomo  
**Categoría:** Loyalty  
**Tags:** loyalty, puntos, pasaporte, redemption, acumulación

## Estructura HTML
```html
<div class="eva-3-pricing-points">
  <img class="pricing-points-logo" src="https://www.staticontent.com/eva-core/assets/logos/0.0.1/loyalty-logo.svg" alt="Logo del programa de Loyalty" />
  <span>Sumás <span class="-eva-3-bold">300 puntos</span></span>
</div>
```

## Características
- Clase base `eva-3-pricing-points`.
- Incluye logo de loyalty a la izquierda y texto libre a la derecha.
- Pensado para acompañar priceboxes (acumulación/canje).

## Cuándo Usar
- Mostrar puntos a sumar o canjear en priceboxes, cards o listados.
- Destacar beneficios de programa de fidelidad cerca del precio.

## Mejores Prácticas
- Mantener el texto corto para una sola línea.
- Evitar repetir logo varias veces en un mismo bloque; ubicar cerca del precio relevante.

## Dependencias
- `logo-loyalty`, `text`.

## Iconos
- No aplica (usa imagen de logo).
